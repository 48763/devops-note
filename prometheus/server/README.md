## 啟用服務

```bash
$ docker run -p 9090:9090 --name prometheus \
    -v $(pwd)/prometheus.yml:/etc/prometheus/prometheus.yml \
    -v $(pwd)/config/:/etc/prometheus/config/ \
    -d prom/prometheus
```

訪問頁面：

```
http://127.0.0.1:9090
```

# Aliyun

## 安裝

```bash
$ wget https://aliyuncli.alicdn.com/aliyun-cli-linux-3.0.59-amd64.tgz
$ tar -xvf aliyun-cli-linux-3.0.59-amd64.tgz -C /usr/local/bin
```

## 登入

```bash
$ aliyun configure
```

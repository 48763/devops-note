# VS Code 實用快捷鍵筆記 (Mac)

整理開發時常用的介面與程式碼整理技巧，提升開發效率。

## 1. 檔案總管 (Explorer) 快速收合

當左側專案結構展開太多資料夾時，可用以下方式快速還原整潔：

* **方法一：按鈕 (最快)**
    * 將滑鼠移到左側「檔案總管 (Explorer)」標題列。
    * 點擊出現的 **「−」 (Collapse Folders)** 圖示即可全部收合。

* **方法二：鍵盤操作**
    * 聚焦在檔案列表，連續按 `Cmd` + `←` (左方向鍵)，可層層收合資料夾。

## 2. 程式碼摺疊 (Code Folding)

針對過長的程式碼檔案，利用摺疊功能讓視野更清晰。VS Code 使用的是組合鍵 (Chord)，需分兩階段按。

| 功能 | 快捷鍵 (Mac) | 記憶口訣 |
| :--- | :--- | :--- |
| **全部摺疊 (Fold All)** | `Cmd` + `K`, `Cmd` + `0` | **K**ontrol -> **0** (歸零全收) |
| **全部展開 (Unfold All)** | `Cmd` + `K`, `Cmd` + `J` | **K**ontrol -> **J** (展開) |
| **依層級摺疊** | `Cmd` + `K`, `Cmd` + `1` | 數字可換 2, 3... 對應層級 |
| **摺疊當前區塊** | `Cmd` + `Option` + `[` | Fold Region |
| **展開當前區塊** | `Cmd` + `Option` + `]` | Unfold Region |

> **Tip**: 如果 `Cmd + K` 組合鍵不習慣，可至 Keyboard Shortcuts 設定搜尋 `Fold All` 自定義按鍵。

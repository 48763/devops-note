# git 

```
$ git clone <repo>
```

```
$ git add .
```

```
$ git commit -m "Update some file."
```

```
$ git push 
$ git push -u origin <branch>
```

```
$ git checkout <branch>
$ git checkout -b <branch>
$ git checkout --track origin/<branch>
```

```
$ git reset --[mixed|soft|hard] master
```

```
$ git branch
$ git branch -a
$ git branch -r
```

```
$ git branch -d <branch>
$ git push origin --delete <branch>
```

```
$ git merge my-branch-name
```
